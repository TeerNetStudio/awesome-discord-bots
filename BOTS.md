## 🤝 Contributing to This File

When adding a new bot to the directory, update both its category page **and** this file.

**Entry format:**

```markdown
#### [Bot Name](bots/{category}/{bot-name}.md)
`{emoji} {Category}` · One-sentence description. {Tags}
```

**Multi-category format** (for bots that span two categories):

```markdown
#### [Bot Name](bots/{primary-category}/{bot-name}.md)
`{emoji} {Category}` `{emoji} {Category}` · One-sentence description. {Tags}
```

**Tags — apply only where accurate:**

| Tag | When to apply |
|-----|--------------|
| ⭐ Featured | Only if the bot is listed as Featured in [README.md](README.md) |
| ✅ Recommended | All listed bots that pass quality review |
| 🆓 Fully Free | Core features require no payment |
| 🔓 Open Source | Source code is publicly available and actively maintained |

**Placement rules:**
- Alphabetical by first letter of the bot's display name
- Numbers sort before letters — bots starting with a digit go at the top of their section
- Update the "Total bots" count in the [Directory at a Glance](#-directory-at-a-glance) table
- Update the "Browse by Category" table if adding a bot to a new or existing category
