# 📜 Logging Bots

> Bots focused on audit trails, moderation logs, message tracking, server event monitoring, and administrative transparency.

**Navigation:** [Main Directory](../README.md) · [All Categories](../README.md#-categories)

Logging bots help administrators monitor activity across their Discord servers by recording important events and actions. They provide visibility into moderation actions, member activity, channel changes, role updates, and other critical server events.

This category covers dedicated logging systems, audit tools, moderation logs, event tracking platforms, and server monitoring solutions.

## 📌 Category Overview

### What this category covers

Bots in this category provide one or more of the following:

* Message logging
* Edit and deletion tracking
* Moderation action logs
* Member join and leave logs
* Role change tracking
* Channel event monitoring
* Audit trails
* Security monitoring
* Administrative transparency
* Event archiving

### What qualifies for inclusion

A bot belongs in this category if it:

* Primarily focuses on logging and monitoring
* Records meaningful server events
* Is actively maintained
* Provides administrative visibility

### What does NOT qualify

The following do not belong in this category:

* Moderation bots with basic logging
* Analytics-only platforms
* Utility bots
* Abandoned projects
* Private monitoring solutions
