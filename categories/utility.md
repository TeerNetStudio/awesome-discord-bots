# 🧰 Utility Bots

> Bots focused on productivity, automation, server management, reminders, information lookup, and everyday Discord enhancements.

**Navigation:** [Main Directory](../README.md) · [All Categories](../README.md#-categories)

Utility bots provide practical tools that improve the day-to-day experience of running and participating in Discord servers. Unlike moderation, economy, or community bots, utility bots focus on convenience, automation, information retrieval, and quality-of-life improvements.

This category covers multipurpose bots, automation tools, reminder systems, information services, server utilities, and productivity-focused Discord bots.

## 📌 Category Overview

### What this category covers

Bots in this category provide one or more of the following:

* Reminders
* Server utilities
* Information lookup
* User tools
* Automation
* Polls and voting
* Timezone tools
* Statistics displays
* Productivity systems
* General-purpose commands

### What qualifies for inclusion

A bot belongs in this category if it:

* Primarily provides utility or productivity features
* Improves server management or member experience
* Is actively maintained
* Offers functionality useful across many server types

### What does NOT qualify

The following do not belong in this category:

* Moderation-first bots
* Economy bots
* Dedicated analytics platforms
* Music bots
* Abandoned utility projects

## 🤖 Bots in This Category

### [Sesh](../bots/utility/sesh.md)

A scheduling and event management bot designed to help communities coordinate meetings, gaming sessions, and events across time zones.

**Best for:** Scheduling and reminders.
**Free tier:** Available.
**Dashboard:** Limited.
**Open source:** No.

---

### [ServerStats](../bots/utility/serverstats.md)

A utility-focused statistics bot that displays server metrics through channel names and live counters.

**Best for:** Live server information.
**Free tier:** Available.
**Dashboard:** Limited.
**Open source:** No.

---

### [YAGPDB](../bots/moderation/yagpdb.md)

While known for moderation, YAGPDB's custom command system makes it one of the most powerful utility and automation tools available.

**Best for:** Advanced automation.
**Free tier:** Generous.
**Dashboard:** Limited.
**Open source:** Yes.

---

### [Carl-bot](../bots/moderation/carl-bot.md)

Beyond moderation, Carl-bot offers automation, reaction roles, reminders, utility commands, and workflow enhancements.

**Best for:** All-around server utility.
**Free tier:** Available.
**Dashboard:** Yes.
**Open source:** No.

→ YAGPDB.

→ Sesh.
