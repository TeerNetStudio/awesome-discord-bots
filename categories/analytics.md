# 📊 Analytics Bots

> Bots focused on server statistics, growth tracking, activity analysis, engagement metrics, and community insights.

**Navigation:** [Main Directory](../README.md) · [All Categories](../README.md#-categories)

Analytics bots help server owners understand what is happening inside their communities. Instead of focusing on moderation or engagement directly, these bots collect, organize, and visualize data about member activity, server growth, channel performance, and community trends.

This category covers bots that provide statistics, dashboards, reports, leaderboards, growth tracking, and operational insights for Discord communities.

## 📌 Category Overview

### What this category covers

Bots in this category provide one or more of the following:

* Server statistics
* Member growth tracking
* Activity monitoring
* Channel analytics
* Engagement reporting
* Historical trends
* User activity reports
* Staff performance metrics
* Automated reports
* Dashboard visualizations

### What qualifies for inclusion

A bot belongs in this category if it:

* Primarily focuses on analytics and reporting
* Provides measurable server insights
* Is actively maintained
* Offers meaningful data beyond Discord's native tools

### What does NOT qualify

The following do not belong in this category:

* Moderation bots with basic stats commands
* Leveling bots
* Logging-only bots
* Utility bots with minimal analytics
* Abandoned projects

## 🤖 Bots in This Category

### [Combot](../bots/analytics/combot.md)

A popular analytics platform focused on community growth, moderation insights, and engagement reporting. Combot provides dashboards and historical tracking for Discord communities.

**Best for:** Community managers and public servers.
**Free tier:** Available.
**Dashboard:** Yes.
**Open source:** No.

---

### [ServerStats](../bots/analytics/serverstats.md)

A statistics-focused bot that displays live server metrics such as member counts, boosts, channels, and activity information through voice and text channels.

**Best for:** Real-time server statistics.
**Free tier:** Available.
**Dashboard:** Limited.
**Open source:** No.

---

### [Statbot](../bots/analytics/statbot.md)

One of Discord's most comprehensive analytics bots. Statbot provides detailed reports, activity tracking, channel statistics, and growth insights through a modern dashboard.

**Best for:** Deep analytics and reporting.
**Free tier:** Available.
**Dashboard:** Yes.
**Open source:** No.

## 🎯 Choosing the Right Bot

| If your priority is... | Consider    |
| ---------------------- | ----------- |
| Deep analytics         | Statbot     |
| Community insights     | Combot      |
| Live server counters   | ServerStats |
| Growth tracking        | Statbot     |
| Large server reporting | Combot      |
| Easy dashboard access  | Combot      |

### Key Questions

**Do you need historical analytics?**
→ Statbot.

**Do you want visual dashboards?**
→ Combot or Statbot.

**Do you want live counters visible in channels?**
→ ServerStats.

**Do you manage a large public community?**
→ Combot.
