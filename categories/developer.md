# 👨‍💻 Developer Bots

> Bots focused on programming, API integration, automation, code utilities, development workflows, and technical communities.

**Navigation:** [Main Directory](../README.md) · [All Categories](../README.md#-categories)

Developer bots provide tools and utilities designed for programmers, bot developers, system administrators, DevOps engineers, and technical communities. They help automate workflows, retrieve documentation, manage repositories, execute utility commands, and integrate development services directly into Discord.

This category covers development assistants, GitHub integrations, code execution tools, API utilities, CI/CD notifications, and programming-focused Discord bots.

## 📌 Category Overview

### What this category covers

Bots in this category provide one or more of the following:

* GitHub integration
* Code utilities
* Documentation lookup
* API testing
* CI/CD notifications
* Deployment monitoring
* Programming tools
* Developer automation
* Self-hosting support
* Technical productivity tools

### What qualifies for inclusion

A bot belongs in this category if it:

* Primarily targets developers or technical users
* Provides programming-related functionality
* Is actively maintained
* Improves development workflows

### What does NOT qualify

The following do not belong in this category:

* General utility bots
* Moderation bots
* AI chatbots without developer focus
* Abandoned developer projects
* Private internal tools

## 🤖 Bots in This Category

### [GitHub Bot](../bots/developer/github-bot.md)

Provides GitHub notifications, pull request updates, issue tracking, release alerts, and repository activity monitoring directly inside Discord.

**Best for:** Development teams and open-source communities.
**Free tier:** Available.
**Dashboard:** Limited.
**Open source:** No.

---

### [DocsBot](../bots/developer/docsbot.md)

A documentation-focused bot that helps developers quickly access technical references, guides, and programming resources.

**Best for:** Learning and documentation access.
**Free tier:** Available.
**Dashboard:** Limited.
**Open source:** No.

---

### [MonitoRSS](../bots/developer/monitorss.md)

A monitoring bot that delivers RSS feeds, development news, project updates, and automated notifications.

**Best for:** Tracking development updates.
**Free tier:** Available.
**Dashboard:** Yes.
**Open source:** No.

---

### [Red-DiscordBot](../bots/developer/red-discordbot.md)

An open-source framework that allows developers to create highly customized Discord bot experiences through extensions and self-hosting.

**Best for:** Developers and self-hosters.
**Free tier:** Free.
**Dashboard:** Community extensions.
**Open source:** Yes.

## 🎯 Choosing the Right Bot

| If your priority is...  | Consider       |
| ----------------------- | -------------- |
| GitHub notifications    | GitHub Bot     |
| Documentation access    | DocsBot        |
| Self-hosting            | Red-DiscordBot |
| Monitoring feeds        | MonitoRSS      |
| Open-source flexibility | Red-DiscordBot |
| Development communities | GitHub Bot     |

### Key Questions

**Do you manage GitHub repositories?**
→ GitHub Bot.

**Do you want maximum customization?**
→ Red-DiscordBot.

**Do you need project update feeds?**
→ MonitoRSS.

**Do you frequently access documentation?**
→ DocsBot.
