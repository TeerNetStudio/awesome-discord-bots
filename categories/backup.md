## 📌 Category Overview

Bots in this category primarily focus on:

- Server backups
- Configuration restoration
- Disaster recovery
- Protection against accidental changes

### Inclusion Criteria

A bot belongs in this category if it:

- Primarily focuses on backups or recovery
- Is publicly available
- Is actively maintained
- Can create or restore server snapshots

### Exclusions

The following do not belong in this category:

- General moderation bots
- Logging-only bots
- Utility bots without backup functionality
- Abandoned projects
