# ✅ Verification Bots

> Bots focused on member verification, CAPTCHA systems, onboarding security, anti-alt protection, and controlled server access.

**Navigation:** [Main Directory](../README.md) · [All Categories](../README.md#-categories)

Verification bots help communities protect themselves from spam, raids, selfbots, automated accounts, and malicious users by verifying members before granting access to the server.

This category covers CAPTCHA systems, verification gates, onboarding workflows, anti-alt protection, join screening, and access control solutions.

Verification is often the first security layer deployed by public Discord communities.

## 📌 Category Overview

### What this category covers

Bots in this category provide one or more of the following:

* CAPTCHA verification
* Human verification
* Anti-bot protection
* Anti-alt systems
* Join screening
* Access control
* Verification roles
* Risk analysis
* Onboarding security
* Community protection

### What qualifies for inclusion

A bot belongs in this category if it:

* Primarily focuses on verification or onboarding security
* Helps identify legitimate users
* Is actively maintained
* Provides meaningful access-control functionality

### What does NOT qualify

The following do not belong in this category:

* General moderation bots
* Security suites where verification is a minor feature
* Utility bots
* Abandoned verification projects
* Private onboarding systems

## 🤖 Bots in This Category

### [Beemo](../bots/security/beemo.md)

A verification and security bot focused on identifying suspicious accounts, reducing abuse, and protecting public communities from malicious users.

**Best for:** Community onboarding security.
**Free tier:** Available.
**Dashboard:** Yes.
**Open source:** No.

---

### [Captcha.bot](../bots/security/captcha-bot.md)

One of the most popular Discord verification bots. Uses CAPTCHA challenges and verification systems to ensure users are legitimate before gaining access.

**Best for:** Human verification.
**Free tier:** Generous.
**Dashboard:** Yes.
**Open source:** No.

---

### [Double Counter](../bots/security/double-counter.md)

A bot designed to identify potential alternative accounts by analyzing joins and user data patterns.

**Best for:** Anti-alt protection.
**Free tier:** Available.
**Dashboard:** Limited.
**Open source:** No.

---

### [Verify](../bots/verification/verify.md)

A lightweight verification platform designed for communities that want a simple onboarding experience.

**Best for:** Basic verification workflows.
**Free tier:** Available.
**Dashboard:** Limited.
**Open source:** No.

### Key Questions

**Do you need CAPTCHA verification?**
→ Captcha.bot.

**Are alternative accounts a major issue?**
→ Double Counter.

→ Beemo.

→ Verify.
