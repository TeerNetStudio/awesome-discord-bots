# 📝 Form Bots

> Bots focused on applications, questionnaires, registrations, surveys, feedback collection, and structured user submissions.

**Navigation:** [Main Directory](../README.md) · [All Categories](../README.md#-categories)

Form bots help communities collect structured information directly inside Discord. They simplify staff applications, event registrations, partnership requests, bug reports, feedback collection, and onboarding questionnaires.

This category covers bots and platforms that specialize in forms, applications, and structured submission workflows.

## 📌 Category Overview

### What this category covers

Bots in this category provide one or more of the following:

* Staff applications
* Moderator recruitment
* Event registrations
* Feedback collection
* Surveys and polls
* Partnership applications
* Bug reports
* Onboarding questionnaires
* Structured submissions
* Approval workflows

### What qualifies for inclusion

A bot belongs in this category if it:

* Primarily focuses on forms or applications
* Supports structured data collection
* Is publicly available
* Is actively maintained

### What does NOT qualify

The following do not belong in this category:

* Basic ticket bots without form support
* Generic moderation bots
* Poll-only bots
* Abandoned projects
* Private application systems

## 🤖 Bots in This Category

### [Formie](../bots/forms/formie.md)

A dedicated Discord forms platform designed for applications, surveys, questionnaires, and structured submissions.

**Best for:** General form management.
**Free tier:** Available.
**Dashboard:** Yes.
**Open source:** No.

### [Helper.gg](../bots/tickets/helper-gg.md)

A support-focused platform with application systems, department routing, and staff recruitment workflows.

**Best for:** Staff applications.
**Free tier:** Available.
**Dashboard:** Yes.
**Open source:** No.

### [Modmail](../bots/tickets/modmail.md)

An open-source support system that can be adapted for structured applications and intake processes.

**Best for:** Support and onboarding.
**Free tier:** Available.
**Dashboard:** Limited.
**Open source:** Yes.

### [Ticket Tool](../bots/tickets/ticket-tool.md)

A ticket platform that supports intake forms and application workflows before creating support channels.

**Best for:** Applications and support intake.
**Free tier:** Available.
**Dashboard:** Yes.
**Open source:** No.

## 🎯 Choosing the Right Bot

| If your priority is... | Consider |
|------------------------|----------|
| Staff recruitment | Helper.gg |
| General forms | Formie |
| Open source | Modmail |
| Application workflows | Ticket Tool |
| Support forms | Modmail |

### Key Questions

**Do you need staff applications?**
→ Helper.gg or Ticket Tool.

**Do you need a dedicated form platform?**
→ Formie.

**Do you prefer open source solutions?**
→ Modmail.
