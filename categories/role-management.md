## 📌 Category Overview

Bots in this category primarily focus on:

- Self-assignable roles
- Reaction roles
- Permission automation
- Member organization
- Access control

### Inclusion Criteria

A bot belongs in this category if it:

- Primarily manages server roles
- Supports automated role assignment
- Is publicly available
- Is actively maintained

### Exclusions

The following do not belong in this category:

- Moderation-only bots
- Verification-only bots
- Utility bots with minimal role features
- Abandoned projects
