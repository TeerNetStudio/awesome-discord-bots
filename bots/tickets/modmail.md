## 🎯 What This Bot Does

Modmail takes a different approach to support compared to traditional ticket bots.

Instead of creating private channels when users need help, members send a direct message to the bot. Staff then reply from a dedicated management server while the user continues chatting through DMs.

This creates a cleaner support experience and prevents support channels from cluttering the main server.

Because Modmail is open source and self-hostable, server owners maintain full control over their support infrastructure and data.
