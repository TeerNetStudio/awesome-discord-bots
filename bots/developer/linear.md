## 🎯 What This Bot Does

Linear helps development teams manage projects, track issues, plan roadmaps, and coordinate work across engineering teams.

Its Discord integration allows teams to receive updates about issue creation, status changes, assignments, comments, project milestones, and workflow events directly inside Discord channels.

Many startups and software teams use Linear alongside GitHub and Sentry to create a complete development workflow covering planning, coding, deployment, and monitoring.

The integration is especially useful for teams that rely on Discord as a central communication platform.
