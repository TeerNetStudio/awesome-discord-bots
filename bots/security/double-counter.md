## 🎯 What This Bot Does

Double Counter is a specialized security bot designed to combat one of Discord's most common problems: alternate accounts.

The bot helps server administrators identify users operating multiple accounts, making it harder for banned users, trolls, and malicious actors to bypass moderation actions.

Many communities use Double Counter alongside moderation bots such as Carl-bot and security platforms such as Wick to create a more secure environment.
