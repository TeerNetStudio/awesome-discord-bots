## 🎯 What This Bot Does

Apollo helps Discord communities organize events, coordinate schedules, and improve member participation.

The bot provides event creation tools, RSVP systems, automatic reminders, recurring events, calendar management, and timezone-aware scheduling. It is particularly useful for gaming clans, creator communities, educational servers, and organizations that regularly host activities.

Instead of relying on manual announcements, Apollo automates event management and keeps members informed about upcoming activities.
