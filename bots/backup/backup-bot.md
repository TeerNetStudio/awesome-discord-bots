## 🎯 What This Bot Does

Backup Bot helps Discord server owners protect their communities against accidental deletions, server damage, misconfigurations, and malicious actions.

The bot can create snapshots of server structures including categories, channels, roles, permissions, emojis, and various server settings. These backups can later be restored when needed.

Many communities use Backup Bot alongside security bots such as Wick and moderation bots such as Carl-bot to improve server resilience.
