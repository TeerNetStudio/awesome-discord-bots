### Adding a bot to this category

1. Confirm the bot meets the [inclusion criteria](#what-qualifies-for-inclusion) defined above
2. Add a bot subsection under [Bots in This Category](#-bots-in-this-category) in alphabetical order
3. Add a row to the [Comparison Table](#-comparison-table) in alphabetical order
4. Update the [Recommended Bots](#-recommended-bots) table if the new bot is a stronger recommendation than an existing entry
5. Create the individual bot page at `bots/{category}/{bot-name}.md` using [`../templates/bot-template.mdbot-template.md`](bot-template.md)
6. Update the `Last Updated` date below

### Updating existing bot information

- Update the bot's subsection entry and comparison table row together — never one without the other
- Update the `Last Updated` date below whenever content changes
- If a bot has been abandoned or its invite link is broken, open an issue before removing it

### Removing a bot from this category

- Do not remove a bot without first opening an issue to confirm it qualifies for removal
- When confirmed, remove the subsection entry, comparison table row, and any recommendation table entries
- Log the removal in [`CHANGELOG.md`](../CHANGELOG.md) with a reason
